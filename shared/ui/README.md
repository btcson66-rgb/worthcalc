# Shared UI Reference

The shared/ui/ folder is a reference library. The actual Astro components live in each site's src/components/. See starter-template/src/components/ for the canonical implementations.

- Header: site-wide top navigation and brand link.
- Footer: legal links, copyright text, and secondary navigation.
- LanguageSwitch: language or locale switcher for multilingual sites.
- FAQ: accessible question and answer section for tool pages.
- RelatedLinks: internal links to related tools or pages.
- ExportButtons: copy and download controls for tool output.
- AdSlot: AdSense slot or inert ad placeholder wrapper.
- Breadcrumb: page hierarchy links for users and search engines.
- SEO: metadata, canonical, Open Graph, and structured data wrapper.
- GoogleAnalytics: GA4 script renderer using the configured Measurement ID.
